# Claude Code

Claude Code is one of the AI coding tools [[Hendrix]] uses day to day. It sits alongside [[Codex]] inside his [[AI-Assisted Development]] workflow. Together they change how much ground a single builder can cover.

## What it is

An AI coding tool that can read, write, and modify code across a project. It acts more like a capable collaborator than a snippet generator.

## How Hendrix uses it

- Building and iterating on [[Duodode]] projects
- Navigating and modifying larger codebases without losing context
- Pairing with [[VS Code]] and [[GitHub]] for real work, not just prototypes
- Running agent-style tasks across files when it saves time
- Offloading the parts of coding that are mechanical so he can stay on direction

## Why it fits his workflow

Hendrix works execution-first. He wants less friction between decision and output. Claude Code compresses that gap. It is another expression of his belief that tools should create real [[Leverage|leverage]], not just feel modern.

## What it unlocks for him

- Higher output per hour, especially on boilerplate
- Serious help with unfamiliar parts of the stack
- A more fluid loop between design, code, and deployment through [[Vercel]]
- More space for the work only he can do: direction, taste, and judgment
